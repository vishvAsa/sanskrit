1. Learn all inter word sandhis (u'll already have an intuition if u've memorized some stotra-s)
2. Understand meanings of vibhakti-s; Focus on declension forms of 'sugaṇ'śabda. Move from halanta-s > ṛ/i/u-kārānta > a/ā-kārānta
3. Understand syntax of a sentence [how vibhakti/liṅga/vacana of nouns interact w saṅkhyā/puruṣa/lakāra of verbs]

---technically done at this point; Now you can parse any sentence, just need to understand meanings of words with a dictionary---

---getting the meanings intuitively comes only with practice, but these are some imp places to begin with this process:---

4. Understand some common auxiliary indeclinables
5. Learn some common suffixes that form nouns
6. Learn some common prefixes which modify verbs
